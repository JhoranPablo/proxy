# 🛒 Mini Proxy Store

Um **mini e-commerce de proxies** desenvolvido para simular a venda de planos de proxy através de uma interface web simples.

O projeto contém páginas de **login, visualização de planos e interface de compra**, focado no desenvolvimento **front-end**.

---

## 📸 Preview

Interface simples com:

* Tela de **login**
* Página principal com **planos de proxy**
* Layout focado em **experiência simples de compra**

## 🚀 Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**

---

## 📂 Estrutura do Projeto

```
/assets        → imagens e recursos visuais
index.html     → página principal da loja
login.html     → página de login
login.css      → estilos da página de login
styles.css     → estilos gerais do site
script.js      → lógica em JavaScript
plano.txt      → informações dos planos de proxy
```
